# myLearningFrontend

Dieses Repository dokumentiert meinen Lernweg im Bereich HTML und CSS.
Alle Lektionen wurden strukturiert aufbereitet, ins Deutsche übersetzt und 
sind über eine einheitliche Navigation miteinander verbunden.

## Repository Evolution

This repository originally contained only the exercises completed during my HTML & CSS course.

After completing the course, I decided to reorganize the entire project:

- translated all lessons into German,
- redesigned the folder structure,
- added consistent navigation,
- improved accessibility (alt/title),
- created a cleaner repository for demonstration purposes.

The original version is available in the `course-original` branch.
